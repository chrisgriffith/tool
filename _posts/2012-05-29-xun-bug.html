--- author: admin categories: app contact: ayound@gmail.com date: '2012-05-29 12:33:43' developer: ayound link: http://sourceforge.net/projects/xunbug/ permalink: /tool/xun-bug/ slug: xun-bug status: publish targetMarket: null thumb: http://phonegap.com/uploads/v2/4f6054f6a8607100.png title: Xun Bug type: tool format: html --- Xun Bug is an application for debug and profiling javascript code. It is a powerful tool supports most of mobile browser such as iPhone,Android,BlackBerry,Windows Mobile and Symbian. There is no need to install plugins in your browser. With the debug tool of Xun bug,you can set breakpoint,watch call stacks,step and resume,watch expressions and so on. With the profile tool of Xun bug, you can get the javascript function call tree ,every call's time and the statistic of function calls.
