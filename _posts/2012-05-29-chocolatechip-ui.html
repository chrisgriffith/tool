--- author: admin categories: app contact: rbiggs@ymail.com date: '2012-05-29 12:29:58' developer: Robert Biggs link: http://www.chocolatechip-ui.com permalink: /tool/chocolatechip-ui/ slug: chocolatechip-ui status: publish targetMarket: null thumb: http://phonegap.com/uploads/v2/4f321532c71e2chocolatechip-ui.png title: ChocolateChip-UI type: tool format: html --- ChocolateChip-UI is a UI framework for mobile Web apps. It provides numerous controls, such as navigation lists, selection lists, deletion lists, switches, popups, popovers, split views, scroll panels, carousels, templates, etc. with minimum scripting. Controls can be implemented with minimal scripting requirements. ChocolateChip-UI comes in three versions for ChocolateChip.js, jQuery and Zepto.js
