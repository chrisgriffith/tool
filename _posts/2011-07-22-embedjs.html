--- author: admin categories: app contact: contact@uxebu.com date: '2011-07-22 20:15:44' developer: uxebu link: https://github.com/uxebu/embedjs/ permalink: /tool/embedjs/ slug: embedjs status: publish targetMarket: null thumb: http://phonegap.com/uploads/v2/4e29a6b663260embedjs-logo-twitter.png title: EmbedJS type: tool format: html --- EmbedJS is a JavaScript framework for embedded devices (mobile phones, TVs, etc.). It takes a different approach than other frameworks by shipping just the code needed for each device. That means, there's less code going over the wire, less code branching at runtime and less memory usage.

EmbedJS is based on the Dojo Toolkit. So to say, it is Dojo's core, highly optimized.
