--- author: admin categories: app date: '2011-03-14 20:09:23' developer: Paul Beusterien link: http://www.mobiledevelopersolutions.com permalink: /tool/mds-eclipse-plugin-for-android-phonegap/ slug: mds-eclipse-plugin-for-android-phonegap status: publish targetMarket: null thumb: http://phonegap.com/uploads/v2/4d769d975d5e9GooGroupPhoneLogoTINY.png title: MDS Eclipse Plugin for Android PhoneGap type: tool format: html --- The MDS plugin extends the Android Development Tools (ADT) and Eclipse JSDT. It includes a project wizard to configure PhoneGap projects, enabling app development and deployment in a few clicks. The wizard offers options to include the jQuery Mobile and Sencha Touch UI frameworks. The editor includes configurable JSLint/JSHint to give real-time feedback on JavaScript typos and style. The included PhoneGap API demo source rapidly gets you up to speed.
