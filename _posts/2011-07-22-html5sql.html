--- author: admin categories: app contact: kenneth.corbett@gmail.com date: '2011-07-22 20:15:13' developer: Ken Corbett link: http://html5sql.com permalink: /tool/html5sql/ slug: html5sql status: publish targetMarket: null thumb: http://phonegap.com/uploads/v2/4e1dd59fb5b3chtml5sql_symbol.png title: html5sql type: tool format: html --- The native HTML5 Web SQL spec has a tricky callback based API that makes it difficult to use. html5sql is a light JavaScript module that makes working with the HTML5 Web Database a whole lot easier. Its primary feature is it's ability to quickly process a batch of SQL statements within a single database transaction, which can even be loaded from a separate text file though ajax. If you need a database for you application, check out html5sql. It will save you a ton of time.
