--- author: admin categories: app date: '2011-01-24 22:46:44' developer: Dave Balmer link: http://joapp.com/ permalink: /tool/jo-html5-framework/ slug: jo-html5-framework status: publish targetMarket: null thumb: /uploads/v2/4d36289c7ab0bjoapp_100.png title: Jo HTML5 Framework type: tool format: html --- Jo is a small HTML5 app framework designed to run on top of PhoneGap. Jo comes to the party with common mobile UI widgets and is completely customizable with CSS3. This open source project is well documented and supported. Jo works with PhoneGap, most mobile browsers, desktop browsers and even Dashboard widgets (see the demo video).
