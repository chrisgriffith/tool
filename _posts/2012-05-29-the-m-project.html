--- author: admin categories: app contact: info@panacoda.com date: '2012-05-29 12:04:49' developer: Panacoda link: http://www.the-m-project.org permalink: /tool/the-m-project/ slug: the-m-project status: publish targetMarket: null thumb: http://phonegap.com/uploads/v2/4e48e4f9c686dlogo_tmp_100x100.png title: The-M-Project type: tool format: html --- The-M-Project is a mobile HTML5 JavaScript application framework to build cross-platform mobile web apps. It includes a rich UI library, data providers for storage abstraction and is built with MVC in mind. It comes together with "Espresso", its little friend, that provides a test server, a built tool and optimizer. The-M-Project apps run on iOS, Android, BlackBerry and webOS.
